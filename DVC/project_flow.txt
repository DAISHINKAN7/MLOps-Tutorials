1. Create a git repo and clone it in local
2. Create code.py and add code to it
3. Do a git add-commit-push before initialising the dvc.
4. pip install dvc
5. Now we can do 'dvc init' (creates .dvcignore, .dvc)
6. Now do 'mkdir s3' (creates a s3 directorys)
7. Now we do "dvc remote add -d myremote s3"
8. Next "dvc add data/"
    Now it will ask to do: ("git rm -r --cached 'data'" and "git commit -m "stop tracking data"")
    Because intially we were tracking data/ folder from git so now we remove it for DVC to handle
9. Again we do 'dvc add data/" (creates data.dvc) then "git add .gitignore data.dvc"
10. Now make changes to code.py to append a new row in data, check changes via "dvc status". 
11. Again - - "dvc commit" and then "dvc push"
